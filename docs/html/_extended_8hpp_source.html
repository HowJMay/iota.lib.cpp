189  /*
190  * This function returns the bundle which is associated with a transaction. Input can by any type
191  * of transaction (tail and non-tail). If there are multiple bundles (because of a replay for
192  * example), it will return multiple bundles. It also does important validation checking
193  * (signatures, sum, order) to ensure that the correct bundle is returned.
194  *
195  * @param transaction The transaction encoded in trytes.
196  * @return an array of bundle, if there are multiple arrays it means that there are conflicting

198  */
